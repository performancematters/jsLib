Binaries

With this package you have obtained all binaries required to embed AnyChart Flash Chart Component to your web site or application, please see the list and decription below. Also we ship a basic sample, that may be used to start study and the development, please do not forget, that there are hundreds of downloadable sample on the AnyChart Site, see Getting Started and Documentation sections.

Binaries

AnyChart Flash Chart Component consists of several files, which you can find in binaries folder:

  • swf/AnyChart.swf - engine SWF file used to render charts in the HTML page with the Flash Player plugin;
  • swf/Preloader.swf - auxiliary file used to show the download progress of the engine SWF file;
  • js/AnyChart.js - library, which makes working with Flash component on the page much easier and provides various ways for embedding and connecting HTML page with chart component, learn more in User's Guide Article.
  • js/AnyChartHTML5.js - AnyChart HTML 5 rendering engine.

These files are necessary and sufficient for your own solution, that uses AnyChart, but you may need additional configuration or data files for some of the sample from articles and documentation.

Basic Sample

In basic-sample folder you can find the very simple HTML page with AnyChart Component chart built-in, you can publish this folder to your website, launch it and see AnyChart Component in action in a matter of seconds. Then please proceed to Getting Started page to go on - making AnyChart work for you requires some attention and careful study of the documentation and help materials.

Optimized SWF Files

Also, in swf folder in binaries, you can find several other SWF files, that can be used to render charts. Each file has almost the same basic features and the two things differs each file from each other: the chart types it can render and the size.

Detailed description of these files can be found at: AnyChart SWF Files Guide.

Flex Library

If want to use Flex version of AnyChart, which is described at Using AnyChart in Flex applications you may obtain AnyChartFlexComponent.swc in binaries/flex/.

Maps

AnyChart comes with the number of amap files, that are used to display Geographic maps, these amap files can be found in binaries/maps/ subfolder. Please copy this folder (or only required amaps) to the proper place, if you want to use maps.
